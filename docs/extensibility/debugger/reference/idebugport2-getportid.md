# IDebugPort2::GetPortId

Gets the port identifier.

## Syntax

### [C#](#tab/csharp)
```csharp
int GetPortId( 
   out Guid pguidPort
);
```
### [C++](#tab/cpp)
```cpp
HRESULT GetPortId( 
   GUID* pguidPort
);
```
---

## Parameters
`pguidPort`\
[out] Returns the GUID that identifies the port.

## Return Value
 If successful, returns `S_OK`; otherwise, returns an error code.
