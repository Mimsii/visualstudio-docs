# IDebugPortSupplierLocale2::SetLocale

Sets the locale for the port supplier.

## Syntax

### [C#](#tab/csharp)
```csharp
int SetLocale(
   ushort wLangID
);
```
### [C++](#tab/cpp)
```cpp
HRESULT SetLocale(
   WORD wLangID
);
```
---

## Parameters
`wLangID`\
Identifier for the locale to set.

## Return Value
 If successful, returns `S_OK`; otherwise, returns an error code.
