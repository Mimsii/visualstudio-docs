# IDebugReference2::GetSize

Gets the size, in bytes, of the value of the reference. Reserved for future use.

## Syntax

### [C#](#tab/csharp)
```csharp
int GetSize ( 
   out uint pdwSize
);
```
### [C++](#tab/cpp)
```cpp
HRESULT GetSize ( 
   DWORD* pdwSize
);
```
---

## Parameters
`pdwSize`\
[out] Returns the size, in bytes, of the value of the reference.

## Return Value
 Always returns `E_NOTIMPL`.
